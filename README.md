# 🚀 Modern Login & Sign Up System

تطبيق واجهة مستخدم (UI) احترافية لصفحات تسجيل الدخول وإنشاء الحساب، مصمم بأسلوب **Glassmorphism** العصري مع نظام تحقق من البيانات (Validation) باستخدام JavaScript.

## ✨ المميزات (Features)
- **تصميم عصري:** استخدام تأثير الـ Blur والخلفيات الشفافة.
- **تفاعلية كاملة:** أزرار وحقول إدخال متفاعلة مع حركة الماوس (Hover & Scale effects).
- **التحقق من البيانات:** رسائل خطأ تظهر فوراً عند إدخال بيانات غير صحيحة أو ترك الخانات فارغة.
- **متوافق مع المتصفحات:** استخدام `normalize.css` لضمان ظهور الموقع بشكل ثابت.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **HTML5:** لبناء هيكل الصفحة.
* **CSS3:** للتنسيق المتقدم (Flexbox, Backdrop-filter, Transitions).
* **JavaScript (Vanilla):** لبرمجة الـ Logic وعمليات الـ Validation.
* **Google Fonts:** خط (Work Sans) لزيادة جمالية الخطوط.

## 📂 هيكل الملفات
- `index.html`: صفحة تسجيل الدخول (الرئيسية).
- `signup.html`: صفحة إنشاء حساب جديد.
- `CSS/`: يحتوي على ملفات التنسيق.
- `javascript/`: يحتوي على ملفات البرمجة والـ Validation.
- `images/`: صور الخلفية المستخدمة.
